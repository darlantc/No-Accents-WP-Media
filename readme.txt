Sanitize the filename of uploaded media in WordPress removing accents and all weird characters

== Description ==

= Português =

Quando você adiciona uma imagem que contém acentos em seu nome no WordPress ela não aparece no Safari (tanto iOS quanto Mac OS).

Este plugin remove acentos e caractéres estranhos do nome de arquivo de qualquer upload feito no WordPress. Assim você não precisa se preocupar em *limpar* o nome dos arquivos antes de fazer o upload.

Especialmente útil para controlar os uploads feitos por **Clientes**.

= English =

When you add an image containing accents in their name in WordPress, that image don't appear in Safari (both iOS and Mac OS).

This plugin removes accents and all weird characters from name of any uploaded file in WordPress. So you don't need to worry about *clean* the name of the files before uploading.

Especially useful for controlling uploads made ​​by **Customers**.

== Installation ==

1. Download the plugin
1. Extract all the files.
1. Upload everything (keeping the directory structure) to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Have Fun!

== Frequently Asked Questions ==

= Where is the settings page? =

This plugin don't have a settings page. Just activate and you are all done!
